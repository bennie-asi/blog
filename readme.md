## 个人博客0.1
### 技术路线
Django + mysql + 模板引擎
### APP
blog，comments，contact
### 外部插件
Faker（模拟数据生成）、pure_pagination（分页）、Markdown（md文档解析）

### The knowledge learned
Django项目搭建、与mysql数据库交互CRUD（ORM）、日志记录器配置、模板引擎技术、创建数据模型（外键、表关系）、迁移操作数据库、Django后台管理
URL路由配置（URL传参）、视图处理、分页处理、RSS订阅等等···
